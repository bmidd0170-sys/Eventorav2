# Eventorav2

## Commands

```bash
pnpm install        # pnpm-lock.yaml is authoritative; package-lock.json is stale
pnpm dev            # next dev ‚Äî kill prior server on port 3000 first
pnpm build          # runs `prisma generate && next build`; ignores TS errors
pnpm lint           # eslint .
npx tsc --noEmit    # actual typecheck ‚Äî build does NOT catch type errors
npx prisma generate # after schema changes
npx prisma db push  # sync schema to Neon/Postgres without creating migration files
```

No test framework is configured.

## Stack

- **Next.js 16** App Router, React 19, TypeScript strict, `@/*` ‚Üí project root
- **Tailwind v4** via `@tailwindcss/postcss` + `tw-animate-css`. All theme in `app/globals.css` (`@theme inline`, no `tailwind.config.js`)
- **shadcn/ui** New York, RSC, lucide. Add: `npx shadcn@latest add <name>`
- **Dark-only**: `<html className="dark">` in `app/layout.tsx:48`
- **Prisma** ‚Äî PostgreSQL via Neon (`DATABASE_URL`). Schema at `prisma/schema.prisma`. Client: `@/lib/prisma.ts` (singleton)
- **Firebase Auth** (`@/lib/firebase.ts`) + **Firebase Firestore** used for social features (connections, requests in `@/lib/connections.ts`)
- **OpenAI** ‚Äî editor AI assistant at `app/api/editor/ai/route.ts` (uses `OPENAI_API_KEY`, falls back to canned replies)
- **Google Maps** ‚Äî loaded lazily in root layout via `NEXT_PUBLIC_GOOGLE_MAPS_API_KEY`
- **Domain lib** modules in `lib/` ‚Äî `connections.ts`, `invitations.ts`, `profile.ts`, `guest-lists.ts`, `notifications.ts`, `email.ts`, `email-templates.ts`, `branding.ts`, `editor-handlers.ts`

## Gotchas

- `next build` ignores TS errors; always verify with `npx tsc --noEmit`.
- Generated by v0.app ‚Äî some boilerplate follows v0 conventions.
- Vercel Analytics renders only in `production` (`app/layout.tsx`).
- Dual persistence: main data (users, events, invitations, guest lists) in PostgreSQL via Prisma; social features (connections, requests) in Firebase Firestore.
- `package-lock.json` is stale ‚Äî use `pnpm` not `npm`.
- `styles/globals.css` is an orphaned shadcn default ‚Äî not imported anywhere; `app/globals.css` is the real stylesheet.
- `scripts/test-prisma.js` can verify DB connectivity (requires `DATABASE_URL`).
- Home screen empty state at `app/(app)/home/page.tsx:416` ‚Äî shows "You have no recent projects."
- `.env` values may have surrounding quotes ‚Äî `lib/firebase.ts` uses `cleanEnv()` to strip them. Ensure other consumers do the same.
- `postcss` pinned via `overrides` in `package.json` ‚Äî don't upgrade independently of Tailwind v4.
