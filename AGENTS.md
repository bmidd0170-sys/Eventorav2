## Commands

```bash
pnpm install        # pnpm-lock.yaml is authoritative; package-lock.json is stale
pnpm dev            # next dev — kill any prior server on port 3000 first
pnpm build          # ⚠️ ignores TS errors (next.config.mjs) — always verify below
pnpm lint           # eslint .
npx tsc --noEmit    # actual typecheck — build does NOT catch type errors
npx prisma generate # after schema changes
npx prisma db push  # sync schema to Neon/Postgres without creating migration files
```

No test framework is configured.

## Stack

- **Next.js 16** App Router, React 19, TypeScript strict, `@/*` → project root
- **Tailwind v4** via `@tailwindcss/postcss` + `tw-animate-css`. All theme in `app/globals.css` (`@theme inline`, no `tailwind.config.js`)
- **shadcn/ui** New York, RSC, lucide. Add: `npx shadcn@latest add <name>`
- **Dark-only**: `<html className="dark">` in `app/layout.tsx:47`
- **Prisma** — PostgreSQL via Neon (`DATABASE_URL`). Schema at `prisma/schema.prisma`. Client: `@/lib/prisma.ts` (singleton)
- **Firebase Auth** (`@/lib/firebase.ts`) + **Firebase Firestore** used for social features (connections, requests in `@/lib/connections.ts`)
- **OpenAI** — editor AI assistant at `app/api/editor/ai/route.ts` (uses `OPENAI_API_KEY`, falls back to canned replies)
- **Google Maps** — loaded lazily in root layout via `NEXT_PUBLIC_GOOGLE_MAPS_API_KEY`

## Gotchas

- `next build` ignores TS errors; always verify with `npx tsc --noEmit`.
- Generated by v0.app — boilerplate may follow v0 conventions.
- Vercel Analytics renders only in `production` (`app/layout.tsx:56`).
- Dual persistence: main data (users, events, invitations, guest lists) in PostgreSQL via Prisma; social features (connections, requests) in Firebase Firestore.
- `package-lock.json` is stale — use `pnpm` not `npm`.
- `styles/globals.css` is an orphaned shadcn default — not imported anywhere; `app/globals.css` is the real stylesheet.
- `scripts/test-prisma.js` can verify DB connectivity (requires `DATABASE_URL`).
- Home screen empty state text at `app/(app)/home/page.tsx:416` — shows "You have no recent projects." when user has no projects (signed in or out).
