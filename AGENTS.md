## Architecture

- **Next.js 16** App Router, React 19, TypeScript strict, `@/*` → project root
- **Tailwind v4** via `@tailwindcss/postcss` + `tw-animate-css`. No `tailwind.config.js` — all config is CSS-based in `app/globals.css` using `@theme inline`
- **shadcn/ui** (New York style, RSC, CSS vars, lucide icons). Add components: `npx shadcn@latest add <name>`
- **Dark-only**: `<html className="dark">` hardcoded in root layout. No light theme.
- **Components**: `@/components/{ui,landing,app,auth}` — ui is the shadcn registry (57 files)
- **Auth**: Firebase Auth via `@/lib/firebase.ts` (exports `auth`, `db`). `.env` has `NEXT_PUBLIC_FIREBASE_*` vars and `DATABASE_URL` (Neon/Postgres, no ORM installed yet)
